Modal Popup - HTML 작성 후 적용

Modal Popup 컨텐츠 영역이 단순 텍스트가 아닌 HTML 코딩이 필요한 경우 아래와 같이 HTML 코드 작성 후 $(".selector").modalPopup(options)로 modalPopup을 적용시킨다.

Options

Name Type Default Description
onOk function function(){} OK 버튼 클릭 시 호출
onCancel function function(){} Cancel 버튼 클릭 시 호출
autoDestroy boolean false 팝업이 닫힐 때 HTML을 remove 시킴
duration number 300 팝업이 열리고 닫힐 때 Fade 효과 적용 시간

Methods

Javascript 
                    
                    // 열기
                    $(".selector").modalPopup("open");

                    // 닫기
                    $(".selector").modalPopup("close");

                    //modal 호출(data-call-modal 이름과 modal의 ID가 같아야함)
                    $('button[data-call-modal]').on('click',function(){
                        var id = $(this).data('call-modal');
                        $('#'+id).modalPopup('open');
                    });

Example

HTML 
<div class="modal" id="modalPopup1">
    <div class="modal_contents">
        로그아웃 후 인증서 발급이 가능합니다.<br>
        확인을 누르면 자동 로그아웃 된 후 인증서 발급이 진행됩니다.
    </div>
    <div class="modal_buttons btn-area">
        <button class="btn-solid_sticky lv2 cancel"><span>취소</span></button>
        <button class="btn-solid_sticky ok"><span>확인</span></button>
    </div>
</div>

Javascript 
$("#modalPopup1").modalPopup({
    onOk: function() {
        console.log('DEMO 1 ok');
    },
    onCancel: function() {
        console.log('DEMO 1 cancel');
    }
});
로그아웃 후 인증서 발급이 가능합니다.

Modal Popup - HTML 미사용

HTML 작성 없이 단순 메시지 팝업으로 호출 할 경우 아래와 같이 modalPopup(options)를 사용한다.
(HTML 코드를 동적으로 생성했다가 팝업을 닫으면 삭제 됨)

Options

Name Type Default Description
title string 알림 타이틀 영역 문구
message string 알림 메시지 메시지 내용 문구
textOkButton string 확인 ok 버튼 문구
textCancelButton string 취소 cancel 버튼 문구
type 'alert', 'confirm' alert alert: ok 버튼만 노출
confirm: ok, cancel 버튼 노출
onOk function function(){} OK 버튼 클릭 시 호출
onCancel function function(){} Cancel 버튼 클릭 시 호출

Example

Javascript 
modalPopup({
    title: "공적연금 추가하기",
    message: "연금개시 연령은 55세~70세 사이로 입력할 수 있습니다."
});
Javascript 
modalPopup({
    title: "공적연금 추가하기",
    message: "기존에 등록한 공적연금이 있습니다. 새로 입력한 공적연금으로 변경하시겠습니까?",
    type: "confirm",
    textOkButton: "변경",
    textCancelButton: "닫기",
    onOk: function() {
        console.log('DEMO 3 ok');
    },
    onCancel: function() {
        console.log('DEMO 3 cancel');
    }
});
